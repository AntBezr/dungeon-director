# Dungeon Director

A web application for Dungeon Masters to run in-person tabletop RPG sessions with the help of digital tools.

Prepare maps, music, NPCs, monsters and scenes in advance, then control everything from a single dashboard while broadcasting selected content to a dedicated player screen.

---

## ✨ Features

### 📚 Campaign Management

* Multiple campaigns
* Scene organization
* Asset libraries
* Session notes

### 🎭 Scene Preparation

* Create reusable scenes
* Assign maps and music
* Place NPCs and monsters
* Configure visibility and starting positions

### 🎮 Game Master Dashboard

* Interactive battle map
* Token management
* NPC and monster visibility controls
* HP tracking
* Session notes
* Music controls
* Scene switching

### 📺 Player Screen

A separate fullscreen view for players.

Displays:

* Active map
* Visible tokens
* NPC reveals
* Story content
* Music

Players never see DM notes, hidden enemies or preparation tools.

### 🏰 Sandbox Support

Not every campaign follows a script.

Quickly switch:

* Maps
* Music
* Artwork
* Story overlays

without leaving the current session.

### 📖 D&D Reference

Powered by an open D&D API.

Search:

* Monsters
* Spells
* Conditions
* Equipment
* Rules

and instantly use them in your session.

## 🎯 Vision

Dungeon Director is not a virtual tabletop.

It is a session control center for Dungeon Masters — a place where preparation, storytelling, maps, music and gameplay come together in a single interface.

Think:

**OBS Studio × Notion × Battle Map for TTRPGs**
